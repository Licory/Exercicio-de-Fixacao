Tags

  1. Tag: <p>
    2. Essa tag é um elemento de paragrafo, também pode ser usada para pular um espaço mas não é muito recomendado.

  2. Tag: <h1>
    3. Normalmente usada para titulos se extendendo em subtitulos usando os numeros de 1 a 6.

  3. Tag: <br>
    4. Usado para pular um paragrafo ou acrescentar um espaço.

  4. Tag: <title>
    5. title é usado para mudar o titulo na guia de uma pagina web.

  5. Tag: <img>
    6. Uma tag de imagem com um atributo obrigatorio de nome "src".

  6. Tag: <input>
    7. É uma tag de interação com o usuario, o permitindo modifica-la.

  7. Tag: <ul>
    8. Uma tag de lista não ordenada, dentro dessa tag temos o <li> como elemento dessa lista.

  8. Tag: <ol>
    9. Diferente da tag anterior, essa é uma tag de lista ordenada tambem contendo <li> como elemento.

  9. Tag: <button>
    10. Cria um layout de botão na pagina que o usuario pode interagir clicando.

  10. Tag: <div>
    11. Define a divisão ou a sessão de um documento html, tambem serve de conteiner para elementos html.
